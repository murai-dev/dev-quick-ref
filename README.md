# errorfix.dev

「開発者が困った瞬間に、3行で答えを出す」英語圏向けリファレンスサイト。

Domain: https://errorfix.dev

## コンセプト

チートシートでも技術ブログでもなく、「特定の問題 → その答え」を1ページで完結させる。

- 1ページ1問題1答え
- ロングテール検索クエリ特化
- 英語のみ、英語圏の開発者が対象
- コピーペースト即使えるコードサンプル

## カバーするカテゴリ（MVP）

- Git
- Docker
- Bash / Shell
- Regex
- npm / Node.js

## 技術スタック

- Vite + TypeScript（静的ビルド）
- Cloudflare Pages（ホスティング）
- GitHub（ソース管理・自動デプロイ）

詳細は `documents/` 以下を参照。
