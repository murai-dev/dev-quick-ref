git / stash-drop

git stash drop — remove a specific stash

Quick Answer

# List your stashes first
git stash list

# Drop a specific stash by index
git stash drop stash@{0}

# Drop all stashes at once
git stash clear

When to use this

Your stash list has grown and you want to clean up entries you no longer need.

Other causes & fixes

Apply a stash and remove it in one step

git stash pop              # applies stash@{0} and drops it

Apply a stash without removing it

git stash apply stash@{1}  # keeps the stash in the list

Show the contents of a stash before dropping

git stash show -p stash@{1}   # full diff of what is in stash@{1}
